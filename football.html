AI/ML Football Analysis

In this project I exposed myself to the applications of computer vision. My goal was to learn about some of the techniques used to generate data from video sources. To achieve this, I utilised the Ultralytics YOLO model and cv2 libraries. The tasks I performed during this project include:

  • Optimising object detection predictions
  • Designing visual overlays that react to object movement
  • Extracting colour data to enable team assignment
  • Designing proximity data to calculate possession statistics
  • Calculating objects' speed and distance while adjusting for camera movement and persepective
